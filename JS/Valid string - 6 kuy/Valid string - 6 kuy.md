<div class="markdown prose max-w-none mb-8" id="description"><p>You are given a sequence of valid words and a string. Test if the string is made up by one or more words from the array. </p>
<h3>Task</h3>

<p>Test if the string can be entirely formed by consecutively concatenating words of the dictionary.</p>
<p>For example: </p>
<pre><code>dictionary: ["code", "wars"]

s1:         "codewars" =&gt;  true  -&gt; match 'code', 'wars'
s2:         "codewar"  =&gt;  false -&gt; match 'code', unmatched 'war'
</code></pre>
<p>One word from the dictionary can be used several times.</p>
</div>

<h3>Завдання</h3>

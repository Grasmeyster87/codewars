<div class="markdown prose max-w-none mb-8" id="description"><p>Given any number of arrays each sorted in <em>ascending order</em>, find the nth smallest number of all their elements.</p>
<p>All the arguments except the last will be arrays, the last argument is <code>n</code>.</p>
<pre><code>nthSmallest([1,5], [2], [4,8,9], 4) // returns 5 because it's the 4th smallest value
</code></pre>
<p>Be mindful of performance.</p>
</div>

<div class="markdown prose max-w-none mb-8" id="description"><p>Дано любое количество массивов, каждый из которых отсортирован в <em>по возрастанию</em>. Найдите n-е наименьшее число среди всех их элементов.</p>
<p>Все аргументы, кроме последнего, будут массивами, последний аргумент — <code>n</code>.</p>
<pre><code>nthSmallest([1,5], [2], [4,8,9], 4) // возвращает 5, потому что это 4-е наименьшее значение
</code></pre>
<p>Учитывайте производительность.</p>
</div>

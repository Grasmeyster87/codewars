<div class="markdown prose max-w-5xl mx-auto overflow-x-auto break-words" id="description"><h3 id="puzzle-1-sum">PUZZLE #1. SUM</h3>
<p>Assignment:</p>
<p>Write a function named <code>sum</code> which performs addition in the way shown below:</p>
<pre><code class="language-javascript"><span class="cm-variable">sum</span>(<span class="cm-number">4</span>)(<span class="cm-number">5</span>)(<span class="cm-number">9</span>)(); <span class="cm-comment">// =&gt; 18</span>
<span class="cm-variable">sum</span>(<span class="cm-number">5</span>)();       <span class="cm-comment">// =&gt; 5</span>
<span class="cm-variable">sum</span>();          <span class="cm-comment">// =&gt; 0</span>
</code></pre>
<pre style="display: none;"><code class="language-ruby"><span class="cm-variable">sum</span>(<span class="cm-number">4</span>)<span class="cm-operator">.</span>(<span class="cm-number">5</span>)<span class="cm-operator">.</span>(<span class="cm-number">9</span>)<span class="cm-operator">.</span>()  <span class="cm-comment"># =&gt; 18</span>
<span class="cm-variable">sum</span>(<span class="cm-number">5</span>)<span class="cm-operator">.</span>()          <span class="cm-comment"># =&gt; 5</span>
<span class="cm-variable">sum</span>()              <span class="cm-comment"># =&gt; 0</span>
</code></pre>
<p>NOTE: Pay attention that the last brackets are left empty to indicate the end of operations.</p>

<div class="w-full panel bg-ui-section"><h3 class="wf-title-alt">Description:</h3><div class="markdown prose max-w-5xl mx-auto overflow-x-auto break-words" id="description"><h2 id="pair-of-gloves">Pair of gloves</h2>
<p>Winter is coming, you must prepare your ski holidays. The objective of this kata is to determine the number of pair of gloves you can constitute from the gloves you have in your drawer.</p>
<p>Given an array describing the color of each glove, return the number of pairs you can constitute, assuming that only gloves of the same color can form pairs.</p>
<h3 id="examples">Examples:</h3>
<pre><code>input = ["red", "green", "red", "blue", "blue"]
result = 2 (1 red pair + 1 blue pair)

input = ["red", "red", "red", "red", "red", "red"]
result = 3 (3 red pairs)
</code></pre>

<div class="w-full panel bg-ui-section"><h3 class="wf-title-alt">Описание:</h3><div class="markdown prose max-w-5xl mx-auto overflow-x-auto break-words" id="description"><h2 id="pair-of-gloves">Пара перчаток</h2>
<p>Зима приближается, пора готовиться к горнолыжному отдыху. Цель этой задачи — определить количество пар перчаток, которые можно составить из имеющихся у вас в ящике перчаток.</p>
<p>Дано множество, описывающее цвет каждой перчатки. Верните количество пар, которые можно составить, предполагая, что пары могут образовывать только перчатки одного цвета.</p>
<h3 id="examples">Примеры:</h3>
<pre><code>input = ["red", "green", "red", "blue", "blue"]
result = 2 (1 красная пара + 1 синяя пара)

input = ["red", "red", "red", "red", "red", "red"]
result = 3 (3 красные пары)
</code></pre>

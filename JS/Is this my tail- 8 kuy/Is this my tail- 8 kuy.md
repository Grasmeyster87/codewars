Some new animals have arrived at the zoo. The zoo keeper is concerned that perhaps the animals do not have the right tails. To help her, you must correct the broken function to make sure that the second argument (tail), is the same as the last letter of the first argument (body) - otherwise the tail wouldn't fit!

If the tail is right return true, else return false.

The arguments will always be non empty strings, and normal letters.

Чтобы помочь ей, вы должны исправить неработающую функцию, убедившись, что второй аргумент (хвост) совпадает с последней буквой первого аргумента (тело) — иначе хвост не подойдет!

Если хвост правильный, верните true, иначе верните false.

Аргументами всегда будут непустые строки и обычные буквы.

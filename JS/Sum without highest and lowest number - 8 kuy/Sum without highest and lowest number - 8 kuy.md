<h2 id="task">Task</h2>
<p>Sum all the numbers of a given array ( cq. list ), except the highest and the lowest element ( by value, not by index! ).</p>
<p>The highest or lowest element respectively is a single element at each edge, even if there are more than one with the same value.</p>
<p>Mind the input validation.</p>
<h2 id="example">Example</h2>
<pre><code>{ 6, 2, 1, 8, 10 } =&gt; 16
{ 1, 1, 11, 2, 3 } =&gt; 6
</code></pre>
<h2 id="input-validation">Input validation</h2>
<p>If an empty value ( <code>null</code>, <code>None</code>, <code>Nothing</code>, <code>nil</code> etc. ) is given instead of an array, or the given array is an empty list or a list with only <code>1</code> element, return <code>0</code>.</p>

<h2 id="task">Завдання</h2>
<p>Підсумуйте всі числа заданого масиву ( cq. list ), крім найвищого та найнижчого елементів (за значенням, а не за індексом! ).</p>
<p>Найвищий або найнижчий елемент відповідно є окремим елементом на кожному ребрі, навіть якщо їх більше одного з однаковим значенням.</p>
<p>Зверніть увагу на перевірку введених даних.</p>
<h2 id="example">Приклад</h2>
<pre><code>{ 6, 2, 1, 8, 10 } => 16
{ 1, 1, 11, 2, 3 } => 6
</code></pre>
<h2 id="input-validation">Перевірка вводу</h2>
<p>Якщо замість масиву задано порожнє значення ( <code>null</code>, <code>None</code>, <code>Nothing</code>, <code>nil</code> тощо), або заданий масив є порожнім списком або списком лише з <code>1</code> елементом, повернути <code>0</code>.</p>

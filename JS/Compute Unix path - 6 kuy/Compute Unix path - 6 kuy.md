<div class="markdown prose max-w-none mb-8" id="description"><p>Your task is simple: combine input parts into a <a href="https://en.wikipedia.org/wiki/Path_%28computing%29#Unix_style" data-turbolinks="false" target="_blank">Unix style path</a>.</p>
<p>The parameters are represented by strings which may contain letters, dots, spaces, slashes and backslashes.</p>
<p>Backslashes must be converted into slashes, spaces must be trimmed, empty parts must be removed and trailing slashes should be removed.</p>
<p>An empty path should be transformed into a slash.</p>
<p>A few example test cases have been added for a better understanding.</p>

<div class="markdown prose max-w-none mb-8" id="description"><p>Ваша задача проста: объединить входные части в <a href="https://en.wikipedia.org/wiki/Path_%28computing%29#Unix_style" data-turbolinks="false" target="_blank">путь в стиле Unix</a>.</p>
<p>Параметры представлены строками, которые могут содержать буквы, точки, пробелы, косую и обратную косую черту.</p>
<p>Обратные косые черты должны быть преобразованы в косые, пробелы должны быть обрезаны, пустые части должны быть удалены, а конечные косые черты должны быть удалены.</p>
<p>Пустой путь должен быть преобразован в косую черту.</p>
<p>Для лучшего понимания было добавлено несколько примеров тестовых случаев.</p>

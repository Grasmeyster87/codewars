<div class="markdown prose max-w-none mb-8" id="description"><h1 id="task">Task</h1>
<p>You are given a string <code>s</code>. Your task is to count the number of each letter (A-Z), and make a vertical histogram as result. Look at the following examples to understand the rules.</p>
<h1 id="example">Example</h1>
<p>For <code>s = "XXY YY ZZZ123ZZZ AAA BB C"</code>, the output should be:</p>
<pre><code>          *
          *
          *
*       * *
* *   * * *
* * * * * *
A B C X Y Z
</code></pre>
<h1 id="rules">Rules</h1>
<ul>
<li>You just need to count the uppercase letters. Any other character will be ignored.</li>
<li>Using <code>*</code> to represent the number of characters. </li>
<li>The order of output is form A to Z. Characters that do not appear in the string are ignored.</li>
<li>To beautify the histogram, there is a space between every pair of columns.</li>
<li>There are no extra spaces at the end of each row. Also, use "\n" to separate rows.</li>
<li>Happy Coding <code>^_^</code></li>
</ul>
</div>

<div class="markdown prose max-w-none mb-8" id="description"><h1 id="task">Задание</h1>
<p>Вам дана строка <code>s</code>. Ваша задача — подсчитать количество каждой буквы (от A до Z) и построить вертикальную гистограмму в качестве результата. Посмотрите на следующие примеры, чтобы понять правила.</p>
<h1 id="example">Пример</h1>
<p>Для <code>s = "XXY YY ZZZ123ZZZ AAA BB C"</code>, результат должен быть следующим:</p>
<pre><code> *
*
*
* * *
* * * * *
* * * * * *
A B C X Y Z
</code></pre>
<h1 id="rules">Правила</h1>
<ul>
<li>Вам нужно только посчитать заглавные буквы. Любой другой символ будет проигнорирован.</li>
<li>Используйте <code>*</code> для обозначения количества символов.</li>
<li>Порядок вывода от A до Z. Символы, не встречающиеся в строке, игнорируются.</li>
<li>Для улучшения внешнего вида гистограммы между каждой парой столбцов ставится пробел.</li>
<li>В конце каждой строки нет лишних пробелов. Также используйте "\n" для разделения строк.</li>
<li>Удачного кодирования! <code>^_^</code></li>
</ul>
</div>

<div class="markdown prose max-w-none mb-8" id="description"><h1 id="be-concise-iv---index-of-an-element-in-an-array">Be Concise IV - Index of an element in an array</h1>
<h2 id="task">Task</h2>
<p>Provided is a function which accepts two parameters in the following order: <code>array, element</code> and returns the <strong>index</strong> of the element if found and <code>"Not found"</code> otherwise.  Your task is to shorten the code as much as possible in order to meet the character count requirements of the Kata.</p>
<p><strong>You may assume that all array elements are unique.</strong></p>
<p>The character limit is <code>60</code> (CoffeeScript), <code>85</code> (JavaScript, Python), <code>161</code> (Java).</p>
</div>

<div class="markdown prose max-w-none mb-8" id="description"><h1 id="be-concise-iv---index-of-an-element-in-an-array">Be Concise IV - Індекс елемента в масиві</h1>
<h2 id="task">Завдання</h2>
<p>Надається функція, яка приймає два параметри в такому порядку: <code>масив, елемент</code> та повертає <strong>індекс</strong> елемента, якщо він знайдений, та <code>"Не знайдено"</code> в іншому випадку. Ваше завдання — максимально скоротити код, щоб відповідати вимогам Kata щодо кількості символів.</p>
<p><strong>Ви можете вважати, що всі елементи масиву унікальні.</strong></p>
<p>Обмеження кількості символів становить <code>60</code> (CoffeeScript), <code>85</code> (JavaScript, Python), <code>161</code> (Java).</p>
</div>

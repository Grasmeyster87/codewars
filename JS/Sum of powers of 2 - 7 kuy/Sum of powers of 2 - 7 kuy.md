<div class="markdown prose max-w-none mb-8" id="description"><h1 id="description">Description</h1>
<p>Given a number <code>n</code>, you should find a set of numbers for which the sum equals <code>n</code>. This set must consist exclusively of values that are a power of <code>2</code> (eg: <code>2^0 =&gt; 1, 2^1 =&gt; 2, 2^2 =&gt; 4, ...</code>).</p>
<p>The function <code>powers</code> takes a single parameter, the number <code>n</code>, and should return an array of unique numbers.</p>
<h2 id="criteria">Criteria</h2>
<p>The function will always receive a valid input: any positive integer between <code>1</code> and the max integer value for your language (eg: for JavaScript this would be <code>9007199254740991</code> otherwise known as <code>Number.MAX_SAFE_INTEGER</code>).</p>
<p>The function should return an array of numbers that are a <strong>power of 2</strong> (<code>2^x = y</code>).</p>
<p>Each member of the returned array should be <strong>unique</strong>. (eg: the valid answer for <code>powers(2)</code> is <code>[2]</code>, not <code>[1, 1]</code>)</p>
<p>Members should be sorted in <strong>ascending order</strong> (small -&gt; large). (eg: the valid answer for <code>powers(6)</code> is <code>[2, 4]</code>, not <code>[4, 2]</code>)</p>

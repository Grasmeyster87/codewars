Create a function that takes three arguments:

    a value to be evaluated for truthiness.
    a function to execute if the first argument is truthy.
    a function to execute if the first argument is falsy.

If the first argument evaluates to truthy, call the second argument (a function). If it evaluates to falsy, call the third argument instead (also a function).

Note: The first argument can be any value, since most languages allow any value to be interpreted as truthy or falsy. In strongly typed languages, however, it will typically be a boolean.


Создайте функцию, принимающую три аргумента:

значение, проверяемое на истинность;
функцию, которая выполняется, если первый аргумент истинен;
функцию, которая выполняется, если первый аргумент ложен;

Если первый аргумент истинен, вызовите второй аргумент (функцию). Если он ложен, вызовите третий аргумент (тоже функцию).

Примечание: Первый аргумент может быть любым значением, поскольку большинство языков позволяют интерпретировать любое значение как истинное или ложное. Однако в строго типизированных языках это обычно булевое значение.

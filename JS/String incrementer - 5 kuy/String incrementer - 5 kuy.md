<div class="markdown prose max-w-5xl mx-auto overflow-x-auto break-words" id="description"><p>Your job is to write a function which increments a string, to create a new string.</p>
<ul>
<li>If the string already ends with a number, the number should be incremented by 1.</li>
<li>If the string does not end with a number. the number 1 should be appended to the new string.</li>
</ul>
<p>Examples:</p>
<p><code>foo -&gt; foo1</code></p>
<p><code>foobar23 -&gt; foobar24</code></p>
<p><code>foo0042 -&gt; foo0043</code></p>
<p><code>foo9 -&gt; foo10</code></p>
<p><code>foo099 -&gt; foo100</code></p>
<p><em>Attention: If the number has leading zeros the amount of digits should be considered.</em></p>
</div>

<br> <hr> <br>

<div class="markdown prose max-w-5xl mx-auto overflow-x-auto break-words" id="description"><p>Ваше завдання — написати функцію, яка збільшує рядок, щоб створити новий рядок.</p>
<ul>
<li>Якщо рядок вже закінчується числом, число слід збільшити на 1.</li>
<li>Якщо рядок не закінчується числом, число 1 слід додати до нового рядка.</li>
</ul>
<p>Приклади:</p>
<p><code>foo -> foo1</code></p>
<p><code>foobar23 -> foobar24</code></p>
<p><code>foo0042 -> foo0043</code></p>
<p><code>foo9 -> foo10</code></p>
<p><code>foo099 -> foo100</code></p>
<p><em>Увага: Якщо число має початкові нулі, слід враховувати кількість цифр.</em></p>
</div>

                <div class="markdown prose max-w-none mb-8" id="description"><p>This kata requires you to write a function which merges two strings together. It does so by merging the end of the first string with the start of the second string together when they are an exact match.</p>
<pre><code>"abcde" + "cdefgh" =&gt; "abcdefgh"
"abaab" + "aabab" =&gt; "abaabab"
"abc" + "def" =&gt; "abcdef"
"abc" + "abc" =&gt; "abc"
</code></pre>
<p>NOTE: The algorithm should always use the longest possible overlap. </p>
<pre><code>"abaabaab" + "aabaabab" would be "abaabaabab" and not "abaabaabaabab"
</code></pre>
